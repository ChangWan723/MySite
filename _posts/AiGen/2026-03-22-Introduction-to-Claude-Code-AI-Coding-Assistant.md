## Getting Started with Claude Code

### Installation
Claude Code can be installed through multiple methods:

```bash
# Option 1: Via npm (recommended)
npm install -g @anthropic-ai/claude-code

# Option 2: Via Homebrew (macOS)
brew install claude-code

# Option 3: Direct download
# Visit https://claude.ai/code for installers
```

### Basic Configuration
Set up Claude Code for your development environment:

```bash
# Initialize configuration
claude config init

# Set your preferred model
claude config set default_model claude-3-5-sonnet

# Create project context file
echo "# Project Context" > CLAUDE.md
```

### Your First Session
Start with simple commands to get familiar:

```bash
# Interactive session
claude

# One-off command
claude -p "Write a Python function to calculate prime numbers"

# Review existing code
claude -p "Review this function for potential improvements" file.js
```
